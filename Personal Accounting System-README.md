Personal Accounting System
A fully functional double-entry bookkeeping application built in Excel/VBA, modeled on professional accounting workflows.
Features

Custom UserForm for transaction entry with dropdown account selection
Debit/credit balance validation with error handling — won't let you post an unbalanced entry
Automatic posting to the general journal, individual account ledgers, and check register
Trial balance sheet updated automatically with each transaction

How to Use

Open the file and navigate to the Journal tab
Click the New Transaction button to launch the entry form
Select accounts from the dropdown lists and enter amounts
Click OK to post — the tool handles the rest automatically

Requirements

Microsoft Excel with macros enabled
