Invoice Generator
A self-contained Excel/VBA invoicing tool designed for small business use. Select parts and services from an inventory tab, generate a formatted PDF invoice, and automatically maintain a running invoice history — all in one file.

Features

Dropdown line item selection tied to an inventory tab containing descriptions, pricing, and tax information
Formulas automatically populate pricing into invoice line item rows based on selection
Customer information UserForm launches on Generate if no customer data is already present
Automatically saves a formatted PDF invoice to the same folder as the tool
Posts completed invoices to a running invoice history tab
Clears the invoice template after generation for immediate reuse

How to Use

Populate the inventory tab with your parts, services, pricing, and tax information
Select line items from the dropdown lists on the invoice template
Press Generate
Enter customer information if prompted
The PDF is saved, opened for review, and the template is cleared automatically

Requirements

Microsoft Excel with macros enabled
A PDF printer driver (included by default in Windows 10 and later)
