Dynamic CSV Importer
A portable Excel tool that automatically detects and imports CSV files from its own folder — no configuration or hardcoded file paths required.
Features

Uses Power Query to detect the tool's own file location dynamically
Automatically scans the folder for available CSV files and populates a dropdown list
One button import — opens the selected CSV, copies all data into the workbook as a structured table, and closes the source file

How to Use

Place the tool in the same folder as your CSV files
Select the file you want to import from the dropdown list
Press Import

Requirements

Microsoft Excel with macros enabled
Power Query (included in Excel 2016 and later)
